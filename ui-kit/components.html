Left-Hand Menu (.ui-lhs-menu)

Left-Hand Menu

Fixed left-hand navigation menu with support for expanded (260px) and collapsed (80px) states. The menu is loaded from shared/menu.html using the includeHTML() utility function from ui-kit/include.js.

Usage: 

<!-- In your HTML body -->
<div id="sidebar-container"></div>

<!-- At end of body, before other scripts -->
<script src="../ui-kit/include.js"></script>
<script>
    includeHTML('sidebar-container', '../shared/menu.html');
</script>

Collapsed State: Add .collapsed class to the menu element (ID: sidebar) to toggle between expanded and collapsed states.

Features:

  • Fixed positioning on left side
  • 260px wide (expanded) / 80px wide (collapsed)
  • Logo with full and collapsed variants
  • Navigation items with icons and labels
  • Active state highlighting for current page
  • Premium upgrade button at bottom
  • Smooth 0.3s transitions between states

Note: The menu content is maintained in shared/menu.html. Update that file to change menu items across all pages.
