# Jiki Design Mockups

This repository contains HTML design mockups for the Jiki project. All designs are static HTML files that demonstrate visual design and layout for implementation by front-end developers.

## Quick Links

- **[Design Catalog](designs/index.html)** - Browse all design mockups organized by category
- **[UI Kit](ui-kit/components.html)** - View all available components and design system elements

## Rules

- No font size below 14px, except for labels with text-transform uppercase and font weight of 500+, which may be 13px.

## Repository Structure

- `designs/` - All design mockups as HTML files
- `ui-kit/` - Design system components and shared styles
- `experimentation/` - Experimental designs and prototypes

## Getting Started

1. Open `designs/index.html` to see all available designs
2. Review `ui-kit/components.html` to understand the component library
3. Check individual design files in `designs/` for specific mockups

For detailed guidance on creating and organizing designs, see `CLAUDE.md`.
